Small Feline, Big Deadline


Four week long C++ coursework project. Developed a 3D physics-driven game featuring AI, networking and collision detection. An initial codebase was provided via tutorial documents, and the expectation is to extend each section of the game to be more functional and useful.

Key features:

• Physics & Collisions
Implemented rigid-body physics for AABB and sphere collision shapes, including collision detection and resolution.
Integrated the physics system into gameplay via bonuses and a rope-bridge.
Position constraints, implemented in the form of a rope bridge.

• AI
Created a state-based enemy AI that can switch between patrolling and chasing.
Grid-based A* algorithm implementation.
Raycasting for line-of-sight.

• Multiplayer Networking
Implemented a client-server model for multiplayer capabilities.
Designed a system to see connected client's current score.
Real-time position updates across the network.

• Game Design & UI
UI system that uses pushdown automata to keep track of it's state.
Maze generation, using data from a file and collectibles and bonuses scattered randomly.
Win/Loss system with the ability to reload the world
