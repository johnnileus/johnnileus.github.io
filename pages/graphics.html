Advanced Graphics for Games


This four-week long coursework project focused on advanced graphics programming techniques. It was structured into two halves with the first introducing these features through documented tutorials, and the second beginning the development of the project and integrating these features into one unified rendering system.

Throughout the project, I implemented the following key features:
• Post-processing, creating fog using depth textures
• Cube-mapping, for improved reflections
• Texture blending: Terrain texture was based on normal
• Shaders: Implemented a custom water shader
• Procedural Terrain generation
• Transparency
• Shadow Mapping
• Real-time lighting
• Skeletal animations
